## 🛠️ Features & Highlights
- **Fully Responsive Design:** Optimized for a flawless experience across mobile, tablet, and desktop devices.
- **Premium UI/UX:** Clean dark mode theme built using sleek typography (`Poppins` & `Playfair Display`) and modern layouts.
- **Dynamic Skill Animations:** High-performance progress bars triggered dynamically using the JavaScript `IntersectionObserver` API when scrolled into view.
- **Project Grid:** Balanced grid showcasing live projects with unified height and alignment layouts.
